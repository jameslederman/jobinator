1. Create `src/jobinator/pipelines/filter.py`:

   Define Pydantic models (NOT SQLModel — these are config objects, not DB tables):

   ```python
   class OnMissing(str, Enum):
       pass_ = "pass"
       fail = "fail"
       estimate = "estimate"

   class SalaryFilter(BaseModel):
       min_usd: int | None = None
       on_missing: OnMissing = OnMissing.pass_

   class LocationFilter(BaseModel):
       allowed: list[str] = ["remote", "hybrid"]
       on_missing: OnMissing = OnMissing.pass_

   class TitleFilter(BaseModel):
       include: list[str] = []
       exclude: list[str] = []

   class FilterConfig(BaseModel):
       salary: SalaryFilter = SalaryFilter()
       location: LocationFilter = LocationFilter()
       title: TitleFilter = TitleFilter()
       company_exclude: list[str] = []

   class FilterResult(BaseModel):
       passed: bool
       reason: str | None = None
   ```

   Implement `apply_hard_filters(job: NormalizedJob, config: FilterConfig) -> FilterResult`:
   - Check exclude lists first (D-09: reject beats match):
     - `title.exclude`: if any keyword (case-insensitive) appears in job.title.lower() -> reject with reason `f"title_exclude: '{keyword}'"`
     - `company_exclude`: if any keyword (case-insensitive) appears in job.company_slug -> reject with reason `f"company_exclude: '{keyword}'"`
   - Salary filter (AND between groups per D-07):
     - If `config.salary.min_usd` is not None:
       - Get effective_salary = `job.salary_max or job.salary_min`
       - If effective_salary is None: check on_missing — if fail, reject with "salary missing (on_missing=fail)"; if pass, continue
       - If effective_salary < min_usd: reject with `f"salary {effective_salary} < floor {config.salary.min_usd}"`
   - Location filter (AND between groups):
     - If `config.location.allowed` is not empty:
       - If job.location_type is None: check on_missing — if fail, reject; if pass, continue
       - If job.location_type not in allowed: reject with `f"location_type {job.location_type!r} not in allowed"`
   - Title include filter (OR within group per D-07):
     - If `config.title.include` is not empty:
       - If no keyword in include list matches (case-insensitive substring) in job.title.lower(): reject with "title matches none of title_include list"
   - If all pass: return `FilterResult(passed=True)`

2. Add `FilterConfig` to Settings loading in `src/jobinator/configs/settings.py`:
